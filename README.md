# Fluke
Simple, multipurpose brute force attack tool.

<h2>How to use:</h2><br>

```
py fluke.py [URL] [USERNAMES FILE] [PASSWORDS FILE]
```

### Example:<br>
```
py fluke.py https://example.com/login usernames.txt passwords.txt
```
